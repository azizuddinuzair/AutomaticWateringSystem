# Circuit Diagrams

This folder contains the circuit diagrams for the automatic watering system.

## Contents
- Wiring diagrams showing connections between the microcontroller, sensors, and pump/relay
- Power supply and grounding layout
- Component pin mappings

## Notes
- Diagrams are intended for reference during assembly and troubleshooting
- Component values and pin assignments may vary depending on the microcontroller used

## Tools Used
- TinkerCad
